# 重大抢课微辅助
方便抢课的微辅助功能，油猴脚本。

可能会更新新功能。

# 目前功能：
+ `I.` 删除提交时的确认提示; 
+ `II.` 添加"重复上次提交"按钮，因延迟提交失败时可以直接重复上次提交的内容（可跨网页、跨域名共用，支持的网址见 `@match`）; 

+ `III.` 弹出选老师窗口中添加"快速选择"按钮，一键选择+确定; 

+ `IV.` 选择选课页面后自动点击检索按钮;

+ `V.` 点击检索按钮时自动输入验证码;
